# Code of Conduct

This project follows the spirit of the
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).
The text below is a concise restatement; the linked document remains the
authoritative reference for any case it does not cover.

## Our Standards

Participants in this project — maintainers, contributors, reviewers, and
issue authors — are expected to:

- Be respectful and constructive in all interactions.
- Assume good faith and ask clarifying questions before escalating.
- Welcome viewpoints from people with different backgrounds and experience
  levels.
- Focus discussions on technical merit, not on the person.
- Accept feedback gracefully and apologise when mistakes affect others.

## Unacceptable Behaviour

The following are not tolerated in any project space (issues, PRs, comments,
chat channels associated with the project):

- Personal attacks, insults, or derogatory comments.
- Discriminatory language or imagery targeting any protected attribute.
- Public or private harassment.
- Publishing others' private information without explicit permission.
- Sustained disruption of conversations.

## Enforcement

Maintainers are responsible for clarifying the standards above and may take
proportionate action (warning, comment removal, temporary or permanent
ban) in response to violations.

To report a concern, open a private security advisory at
`https://github.com/moukrea/agentify-marketplace/security/advisories/new` or
contact a maintainer directly. Reports are kept confidential to the extent
practical.

## Scope

This Code of Conduct applies within all project spaces and in public spaces
when an individual is representing the project (for example, using an
official project channel, posting via an official account, or acting as an
appointed representative).
